Software used

I was using Debian sid, other versions may suffice.

gqrx

gqrx website - install from debian package

install

apt-get install gqrx

inspectrum

inspectrum website - build from git, version in debian (0.1.6) was too old to find symbol rate

dependencies

  1. cmake (debian package)
  2. qt5-default (debian package)
  3. pkg-config (debian package)
  4. liquid-dsp - build from git, not in debian

install

apt-get install build-essential qt5-default pkg-config
git clone https://github.com/jgaeddert/liquid-dsp.git
cd liquid-dsp.git
cmake
sudo cmake install
# my system had the wrong perms in /usr/local/...
chmod a+r /usr/local/lib/*liquid*
chmod -R a+rX /usr/local/include/liquid

cd ..
git clone https://github.com/miek/inspectrum.git
cd inspectrum
cmake
sudo cmake install
# my system had the wrong perms in /usr/local/...
chmod a+rx /usr/local/bin/inspectrum

rf-car

rf-car website (build from git, not in debian)

dependencies

  1. libsdl2-image-dev (debian package)
  2. libhackrf-dev (debian packages)

install

apt-get install libsdl2-image-dev libhackrf-dev
git clone https://github.com/rgerganov/rf-car.git
cd rf-car
make
sudo make install

proxmark3

proxmark3 website (build from git)

dependencies

  1. libreadline-dev
  2. gcc-arm-none-eabi
  3. libusb-dev
  4. ncurses-dev

install

apt-get install p7zip git build-essential libreadline-dev gcc-arm-none-eabi libusb-dev ncurses-dev pkg-config wget
git clone https://github.com/Proxmark/proxmark3.git
cd proxmark3
make client
